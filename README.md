# 🎨 Paint Brush Application

A **Paint Brush** application built using **C# Windows Forms (.NET Framework)** that allows users to draw and paint with various tools, colors, and brush sizes.  

## ✨ Features

- 🖌️ **Drawing Tools:** Pencil, Brush, Spray  
- 📏 **Shapes:** Rectangle, Circle, Square, Line  
- 🎨 **Fill & Color Picker:** Choose custom colors  
- 🔄 **Undo/Redo Functionality**  
- 🧹 **Clear Canvas Option**  
- ✏️ **Adjustable Brush Size**  
- ❌ **Eraser Tool**  
- 📁 **Open and Save Drawings**  
- 🔲 **User-Friendly UI**

## 🛠️ Installation Guide
1️⃣ **Clone the Repository**
2️⃣ **Open in Visual Studio**
Open Visual Studio
Click on "Open a project or solution"
Navigate to the Paint Brush Application folder and select the .sln file
3️⃣ **Build & Run**
Press F5 or click on Start to run the application

## 🚀 Usage
1️⃣ **Select a Drawing Tool**
Pencil: Freehand drawing
Brush: Smooth strokes
Spray: Spray paint effect
Eraser: Erase any part of the drawing
2️⃣ **Draw Shapes**
Rectangle, Circle, Square, Line: Click and drag to draw
3️⃣ **Adjust Settings**
Choose Colors using the Color Picker
Change Brush Size using the numeric selector
4️⃣ **Modify Your Drawing**
Undo/Redo actions
Clear the Canvas to start fresh
Save your work for later use
